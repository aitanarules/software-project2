On this page, you will be able to visualize the historical data. Just click on the "Upload" button and choose the JSON file you previously saved.

Additionally, if you want to visualize the historical data on the same scale to ease comparison, click on the "Upload Scaled" button. The scaling method used was min-max scaling, and the resulting values will vary between [0, 1].

x' = (x - xmin) / (xmax - xmin)

Where the min and max values refer to the theoretical ones of each variable. This way, we can observe atypical values (those which are greater than 1 or less than 0).
